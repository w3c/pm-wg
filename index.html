Publishing Maintenance Working Group

This is the repository used for the everyday operation of the W3C Publishing Maintenance Working Group. See also charter of the Working Group for further details.

The list or other github repositories, maintained by this Working Group, is also available on-line.

Minutes

Minutes are taken using the W3C Bots on IRC, namely RRSAgent and Zakim. See also the separate page on IRC for further details on how to access the W3C IRC servers. The PM Working Group uses the #pmwg IRC Channel. The minutes of the meetings, including the minutes of the WG Task Forces, are stored on the minutes folder on this repository under the name YYYY-MM-DD[-TF].html (where the optional [-TF] stands for -a11y, -ann, -dc, or -fxl for the accessibility, annotation, digital comics, and fixed layout task forces, respectively). An index page displaying the list of all minutes with their respective table of contents, as well as a resolution page listing the formal resolutions passed by the Working Group, are generated automatically.

Contributing to the Publishing Maintenance Working Group Repositories

Use the standard fork, branch, and pull request workflow to propose changes to the repository contents. Please make branch names informative—by including the issue or bug number for example.

Please read CONTRIBUTING.md, about licensing contributions.
